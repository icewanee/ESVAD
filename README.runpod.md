# Running on Runpod

1. Register at Runpod.io
2. Add some money, you can start with $20 and add when balance runs low.
3. You need to add container registry auth as your free Dockerhub account. If not Runpod's shared account has not enough quota to pull images from Docker.
4. Uses RTX4090 instance with the docker image `docker.io/icewanee/esvad`, use 200GB disk space for SARD dataset and 400GB disk space for Juliet dataset.
